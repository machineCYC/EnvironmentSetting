# Cpp

- compile the cpp code
    - g++ <cpp script name> -o <complie file name>, e.g. g++ test.cpp -o test

- run cpp code
    - ./<complie file name>, e.g. ./test


- get variable memory position
    - &<variable name>

- get memory position value
    - *<variable name with memory position>
